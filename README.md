# WorkWatch 🔍

**Branch-level workplace transparency powered by AI.**

## What it does
WorkWatch helps job seekers make informed career decisions by surfacing anonymous, verified reviews at the branch level — not just company-wide. Reviews are transformed into Manager Archetypes like Micromanager, Burnout Driver, or Mentor, giving candidates real intelligence before they accept an offer.

## Features
- 🗺️ Interactive US map with branch-level pins
- 📊 Risk scoring per branch (0–100)
- 🏷️ AI-powered Manager Archetype classification
- ✅ Verified reviews via SHA-256 email hashing
- 🔒 Anonymous submissions — no names, ever

## Built With
- React + Vite
- Tailwind CSS
- Leaflet.js
- OpenAI Codex

## Live Demo
https://workwatch-swtz.vercel.app

## Purpose
WorkWatch reduces information asymmetry between large corporations and individual workers — supporting equitable employment decisions nationwide.
